# コンパイルの仕方

自分でコンパイルしたい方に  
Keysクラスを作成し、アプリケーションのConsumerKeyとConsumerSecretを記述する.

```C#:Keys.cs
namespace Suiren
{
    class Keys
    {
        public static string ConsumerKey = "Key_hogehoge";
        public static string ConsumerSecret = "Secret_fugafuga";
    }
}
```
